== Description ==

Add checkout steps at your Woocommerce shop for a better shopping experience.

From the moment the user add something to the cart, a 4-steps bar appears bellow the theme header, 
witch will guide the user throwght all the shopping process.

The steps are: Choose products - Cart - Checkout - Done

Each step comes with an icon to ilustrate it. For mobile or small resolutions, only the icons will be showed.

Each step are at the same time link to each shopping step process.

Languages avaliables: English, Spanish and French

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/syw-woocommerce-simple-checkout-steps` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. That's all!, the plugin must be working


== Frequently Asked Questions ==

= It works with my theme?  =

Yes, as long as your theme has a standard Header. Obviosly you need Woocommerce for make it works.

= How can I change the default colors?  =

Into de plugin directory, go to "/css/style.css" and manually change the colors (in hexadecimal).
(Setup from WP's back-end in next release)
